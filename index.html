Sistema de Irrigação Automático

O projeto tem como objetivo a criação de um sistema de irrigação por gotejamento automatizado para residências, com o intuito de fornecer a quantidade ideal de água para plantas, de forma que as mesmas possam se desenvolver de maneira adequada, evitando o desperdício de água e dispensando o uso de trabalho manual. Para essa finalidade, foi utilizado um sensor de umidade (higrômetro) para medir a quantidade de água no solo, a plataforma Arduino para monitorar o fornecimento de água às plantas, um módulo relé para controlar o momento certo para irrigação, dentre outros componentes auxiliares. Além disso, foi feita a simulação desse sistema na ferramenta Fritzing. Como resultado, obteve-se um sistema de irrigação automática de baixo custo, o qual pode ser configurado para umidade ideal de preferência do usuário ou utilizando o valor padrão.
